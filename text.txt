Список команд для работы с гитом.


git init: Создает новый репозиторий Git.
git clone: Клонирует репозиторий Git на локальный компьютер.
git add: Добавляет файлы в индекс Git.
git commit: Фиксирует изменения в репозитории Git.
git status: Показывает текущее состояние репозитория Git.
git push: Отправляет изменения в удаленный репозиторий Git.
git pull: Получает изменения из удаленного репозитория Git.
git branch: Показывает список веток в репозитории Git.
// git branch -M master
git checkout: Переключает ветки в репозитории Git.
git merge: Объединяет ветки в репозитории Git.
git remote.
// git remote add origin *link*.
